# AGAS Bid Tab Analyser & Pricing Tool

AI-powered tool for analysing historical government bid tabulations and generating competitive price suggestions.

## Features
- **Upload bid tab PDFs** — AI extracts all competitor pricing automatically
- **Bid library** — stores and summarises all historical bid data
- **Price Suggester** — AI recommends competitive price ranges for new bids
- **Competitor Analysis** — tracks win rates and pricing patterns by vendor

## Live Tool
👉 **[Open the Bid Tab Analyser](https://14Asif.github.io/bid-tab-tool/)**

## How to use
1. Open the tool via the link above
2. Go to **Upload Bid Tabs** — drag and drop any PDF bid tabulation
3. AI automatically extracts agency, competitors, prices and winner
4. Go to **Price Suggester** — enter new bid details to get a price recommendation
5. Go to **Competitor Analysis** — see win rates and pricing patterns

## Adding more bid tabs
The more historical bid tabs uploaded, the more accurate the pricing suggestions.
Target: 20+ bid tabs across different agencies and years.

## Notes
- Data is saved in browser localStorage — persists between sessions on the same device
- Works with any government bid tabulation PDF format
- Pre-loaded with City of Austin banner bid (January 2021) as baseline data
